## Wiadomość do Osoby 2
Siema, dla części 2 bierzesz metody drzewiaste: **Decision Tree** i **Random Forest**.
Masz już gotowy starter w repo, więc niczego nie budujesz od zera.

Twoje zadanie:
- uruchomić i sprawdzić eksperymenty dla:
  - `Decision Tree` z parametrem `max_depth = [3, 5, 10, 20]`
  - `Random Forest` z parametrem `n_estimators = [50, 100, 200, 300]`
- zrobić to osobno dla:
  - `classification_divorce`
  - `regression_children`
- oddać wyniki CSV, wykresy i krótki opis co wyszło najlepiej.

Odpalasz:
```bash
python run_um_experiments.py --owner person2
```

Nie ruszasz:

- tasków,
- struktury projektu,
- plików od SSN.

Najpierw przeczytaj:
